About Invoicely

                            
Invoicely is a complete Invoice Extraction & Quality Control Service
                                that automatically extracts structured data from PDF invoices, validates the extracted
                                information against comprehensive business rules, and provides quality control reports
                                through multiple interfaces.

                        


                        

                            
Core Components

                            
    
                                
  • PDF Extraction Module - Extracts structured data from PDF invoices
                                    using text parsing and pattern matching with support for multiple invoice formats
                                
    • 
                                
  • Validation Core - Validates extracted data against completeness,
                                    format, business logic, and anomaly detection rules with a 100-point scoring system
                                
    • 
                                
  • CLI Tool - Command-line interface for batch processing invoices
                                    (extract, validate, or full-run)
    • 
                                
  • HTTP API - RESTful API built with FastAPI for programmatic access
                                    to all features
    • 
                                
  • Web UI - Full-featured web application for interactive invoice
                                    processing and management
    • 
                            

                        


                        

                            
Hybrid AI Architecture

                            
This service uses a multi-layered "Hybrid AI" approach to ensure the highest possible
                                extraction accuracy:

                            
    
                                
  • 1. Google Document AI (Primary): Uses the specialized
                                    `pretrained-invoice-v1.3` model for high-fidelity extraction of tabular data and
                                    entities.
    • 
                                
  • 2. Gemini 2.5 Flash (Fallback): A multimodal model that "reads" the
                                    document visually to capture messy layouts, handwritten notes, or images that
                                    standard OCR misses.
    • 
                                
  • 3. Regex Extractor (Backup): A fast, rule-based fallback for simple
                                    text-based PDFs when AI services are unavailable.
    •

    Completeness: Checks for Invoice Number, Date, Vendor, and Total.
    
                        
    Arithmetic: Ensures `Subtotal + Tax ‚âà Total Amount` within a small
                            tolerance.
    
                        
    Logic: verifying `Due Date` >= `Invoice Date`.

    How to Use
    
                        
    Via Web UI:
    
                        
      
                            
    • Go to Upload and select your Extraction Mode.
      • 
                            
    • Drag & Drop a PDF invoice.
      • 
                            
    • Review the extracted data and validation score immediately.
      • 
                        
    
                        
    Via API: Full Swagger docs at /docs.
    
                        
    Via CLI:
    
                        
      
                            
    • python backend/cli.py extract "invoice.pdf" --method auto
      • 
                            
    • python backend/cli.py process-batch "./folder" --output results.json
